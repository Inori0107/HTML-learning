01.Sass(SCSS) 預前處理器

Sass 目的減少 CSS 的重複性工作，需編譯成原生 CSS 才能使用。(瀏覽器看不懂 Sass(SCSS))

Sass 可以使用變數、迴圈、條件判斷、函式、mixins、嵌套等功能...。

02.變數

變數用 $ 宣告 大括號內是區域變數 最外層是全域變數

變數的底線 _ 和連接線 - 是相同的 ex: bg_dark 和 bg-dark 是一樣的

使用變數 可以維持網站的一致性 只需修改變數的值

05.extend %

使用 % 定義名稱 引用時使用 @extend

繼承可以定義一組相同的樣式 供其他選擇器使用

繼承可形成 群組選擇器 使編譯出去的樣式碼減少
